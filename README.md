# File Organizer (Python)

A Python automation tool that organizes files in a directory into categories like Images, Videos, Documents, and Others.

## 🚀 Features
- Automatically detects file types
- Organizes files into folders
- Handles duplicate file names safely
- Accepts user input for folder path
- Provides summary of moved files

## 🛠️ Technologies Used
- Python
- os module
- shutil module

## ▶️ How to Run

1. Run the script:
   python file_organizer.py

2. Enter the folder path when prompted

## 📌 Example

Before:
folder/
  photo.jpg
  video.mp4
  notes.pdf

After:
folder/
  Images/
  Videos/
  Documents/
